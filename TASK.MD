โจทย์เบื้องต้นตอนนี้คือ 

setup frappe framework ด้วย docker 
สร้าง new site แล้วลง ERPNext
ลองเล่น module ต่างๆ ตาม course ที่ส่งให้ดู
ลอง customize ERPNext ด้วยการสร้าง custom app https://docs.frappe.io/framework/user/en/tutorial/create-an-app
customize field ของ DocType ใน ERPNext
สร้าง doctype ใหม่
seed data ด้วยการใช้ fixture
แก้ไข form logic ด้วยการ inject custom javascript เข้าไปในหน้า form ของ doctype
สร้าง dashboard (ใน ERPNext เรียกว่า workspace)
